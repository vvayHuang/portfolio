Stratum - AI 職涯顧問

nuxtvuetailwindsupabasegemini

專案簡介

Stratum 是一個基於 Nuxt 3 構建的 AI 職涯顧問應用程式。它利用 Google Gemini API 根據使用者的興趣生成創意的職涯點子。專案使用 Tailwind CSS 進行樣式設計，GSAP 製作動畫，並整合 Supabase 處理使用者認證與資料儲存。功能包含社群分享、個人點子管理以及詳細的職涯指南生成。(提供訪客測試帳號)

開發技術

  • 框架：Nuxt 3, Vue 3, Tailwind CSS

  • 後端與資料庫：Supabase (PostgreSQL, 身份驗證, RLS 策略)

  • AI 整合：Google Gemini API (伺服器端處理)

  • 動畫特效：GSAP

負責項目

  • 設計可擴展的系統架構：Nuxt 3 客戶端 ↔ 伺服器端 API 代理 ↔ 外部服務 (Gemini/Supabase)，確保職責分離

  • 設定 Supabase Auth 與 RLS 策略保護資料，並管理全端應用程式的 Session 狀態

  • 效能優化：利用 Nuxt SSR、資源壓縮與高效的 GSAP 動畫，達成優異的 Lighthouse 評分

  • 使用 Tailwind CSS 開發 Material Design 3 主題，確保無障礙設計與響應式排版

主要功能

  • AI 智能建議：根據使用者興趣生成個人化的職涯路徑

  • 伺服器端串流指南：即時生成包含步驟的詳細職涯指南

  • 社群與個人管理：分享創意並管理個人的提示詞紀錄

挑戰與成果

  • 利用串流處理技術整合 AI 回應，提供即時且流暢的使用者體驗

  • 透過伺服器端處理所有敏感 API 請求，確保金鑰安全不外洩

學習收穫

  • 深入理解 Nuxt 3 伺服器引擎 (Nitro) 與安全 API 架構設計

  • 掌握生成式 AI 串流與 Vue 3 響應式狀態的整合技巧

Buddy House

figmamaterialdesignbrandingrecraftheadlessui

專案簡介

Buddy House 是一個模擬餐廳形象設計的官網，傳遞「來到家裡吃飯喝酒」的溫馨理念，營造輕鬆友善的用餐氛圍。設計稿以 Figma 製作，透過 Material Theme Builder 快速建立綠色（靈感來自捷運綠線）的配色方案和字體系統，確保整體視覺的一致性與品牌辨識度。此外，善用 Figma 的 Variables 功能，靈活調整設計元素，提升設計效率與彈性。網站中的插畫採用 AI 工具 Recraft 生成，打造獨特且符合品牌形象的視覺效果。此專案結合了設計與技術應用，展現了我在品牌形象設計、工具運用以及氛圍營造上的專業能力。

開發技術

  • 前端技術： Vue.js, Vite, Tailwind CSS, Headlessui

  • 設計工具： Figma, Material Design Builder

  • 插畫或圖片: Recraft AI

負責項目

  • 使用 Figma 設計網站 Wireframe，並透過 Material Design Builder 建立設計系統，確保整體視覺風格一致

  • 透過 Material Design Builder 快速建立字型與顏色變數

  • 開發餐廳網站，實作動態菜單與線上訂位功能

  • 使用 Tailwind CSS 和 Headlessui 進行版面調整與樣式統一

  • 採用 Recraft AI 生成插畫與圖片，提升網站視覺吸引力

主要功能

  • 清楚的菜單內容展示，方便使用者瀏覽餐點資訊

  • 線上訂位功能，讓使用者能夠快速預約

  • 響應式設計，確保桌機與行動裝置皆有良好瀏覽體驗

挑戰與成果

  • 優化 UI/UX 以提升使用者體驗，並確保 GSAP 動畫不影響網站效能

  • 成功打造具品牌一致性的樂團官網，並透過 GSAP 增強視覺互動，提高用戶黏著度

學習收穫

開發技術

形象官網

生活唱片

vitebootstrap

專案簡介

生活唱片是一個模擬線上唱片行的網站，專為音樂愛好者打造。網站提供多樣化的唱片類型，使用者可以輕鬆瀏覽專輯封面、介紹與歌曲內容，深入了解每張專輯的特色。此外，若有興趣的唱片，使用者可透過填寫表單通知功能，進一步進行聯繫或預訂。網站也設有音樂新聞區，提供最新的音樂相關資訊，增強用戶的互動與參與感。開發過程中採用 Bootstrap 框架，實現了整潔的版面設計與響應式布局，確保網站在不同裝置上的良好體驗。這項作品展現了我對音樂主題網站的規劃能力與前端開發的實踐經驗。

開發技術

  • 前端技術： Bootstrap

負責項目

  • 使用 Bootstrap 開發響應式網站，並應用多種 Bootstrap 元件

  • 客製化 SCSS 來優化 CSS 檔案大小，提升載入速度

  • 設計與開發音樂新聞區、代理唱片區、最新發行與唱片介紹頁面

  • 提供清楚的訂購方式與聯絡資訊，提升用戶購買體驗

主要功能

  • 清楚的唱片行介紹，幫助使用者快速了解品牌與服務

  • 即時更新的音樂新聞，讓使用者掌握最新音樂動態

  • 代理唱片區，展示與代理品牌合作的音樂作品

  • 最新發行專區，提供即將發行或近期上架的唱片資訊

  • 唱片介紹與詳細資訊，提升使用者購買決策體驗

  • 訂購說明頁面，提供完整的購買流程與客服聯絡方式

挑戰與成果

  • 第一次使用 Bootstrap，需熟悉其元件與客製化 SCSS 來優化網站樣式與效能

  • 成功利用 Bootstrap 建立響應式網站，並透過 SCSS 優化 CSS 大小，使網站載入更快

學習收穫

  • 熟悉 Bootstrap 各類元件的應用與響應式開發

  • 掌握 SCSS 的客製化技巧，優化 CSS 檔案大小與效能

  • 提升對於音樂相關網站資訊架構的設計與規劃能力

專業技能

前端開發

  • 熟悉 Vue 3 (Composition API) 與 Nuxt 3，具備建構 SSR 及全端應用的開發經驗。

  • 熟悉使用 Pinia 進行狀態管理及 Vue Router 進行路由規劃。

  • 熟悉運用 Tailwind CSS 與 Bootstrap 5 等現代 CSS 框架，並結合 SCSS 編寫可擴展且易於維護的樣式。

  • 具備串接第三方 API（如 Google Gemini API）及處理非同步資料流的實作經驗。

  • 熟悉使用 GSAP 製作流暢動畫與基礎的互動式使用者介面。

設計與 AI 工具

  • 熟悉使用 Figma 進行 UI/UX 設計與原型製作，專注於建立可擴展的設計系統（Design System），並確保設計稿與前端開發之間的高度還原與整合。

  • 熟悉整合 Google Gemini API 等生成式 AI 技術，並運用 Gemini CLI 等工具輔助程式開發。

  • 擅長運用 Recraft AI 等 AI 視覺生成工具，產出高品質的元件素材與插畫。

開發與分析工具

  • 熟悉 Vite 等現代前端建置工具，以及 Git/GitHub 等版本控制系統。

  • 熟悉設定與使用 Microsoft Clarity、Google Analytics 等使用者行為分析工具。
